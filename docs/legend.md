# Legend / Data Dictionary

This page explains the columns used in the NBA dataset and notebooks.

## Identifiers & metadata

| Column | Description |
|---|---|
| `PLAYER_ID` | Unique identifier for each player |
| `SEASON_ID` | Season identifier for the record |
| `LEAGUE_ID` | League identifier (e.g., NBA) |
| `TEAM_ID` | Unique identifier for the team |
| `TEAM_ABBREVIATION` | Team abbreviation |
| `full_name` | Player full name |
| `Year` | Year of the data (season year) |

## Player info

| Column | Description |
|---|---|
| `PLAYER_AGE` | Player age |

## Games played

| Column | Description |
|---|---|
| `GP` | Games played |

| `MIN` | Minutes played |
